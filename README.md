Study project


## Building

Create environment (Windows)
```
virtualenv .venv
cd .venv/Scripts
activate
pip install -r ../../requirements.txt
```

## Start the project

```
python main.py
```

## Development

There is libs for development
(Use pip from .venv environment)

```
pip install requirements.dev.txt
```
